Como editar os preços

No final do arquivo existe uma constante chamada CONSORCIO_CATALOGO. É nela que você troca nome do modelo, categoria, valor de crédito, parcela inicial, prazo, tipo de plano e link de simulação.

Para adicionar um novo modelo, basta duplicar um objeto existente e editar os campos. O site atualiza automaticamente os cards, filtros e contadores.

Fluxo ideal de atualização

  • Abra a tabela pública do Consórcio Honda.
  • Copie crédito e parcela do modelo desejado.
  • Atualize o objeto correspondente no HTML.
  • Envie o arquivo para o GitHub.
  • Publique no GitHub Pages.

Observações importantes

Este layout foi pensado para venda consultiva: visual limpo, leitura rápida, destaque de parcela e forte chamada para ação. Você pode personalizar WhatsApp, biografia, endereço da concessionária e depoimentos sem alterar a estrutura principal.

Valores e condições podem sofrer reajustes pela administradora, alterações do modelo referenciado e atualização de taxas administrativas e seguro prestamista, quando contratado.
